## 轉寫

| 擬音 | 轉寫 |
| :---: | :---: |
| ə | y |
| ʔ | x 或不寫 |
| ɣ | xx 或 f 或 v |
| ɣ 尾 | 可作 g |
| ʰ | h |
| i、j | 可以混淆 |
| u、w | 可以混淆 |
